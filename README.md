# Welding Defect Object Detection

This project focuses on detecting welding defects using **computer vision and deep learning object detection techniques**. The model analyzes welding images and identifies defects by drawing bounding boxes around defective regions.

## Features
- Automated welding defect detection
- Object detection using deep learning models
- Bounding box localization for defects
- Helps improve industrial quality inspection

## Technologies Used
- Python
- PyTorch
- OpenCV
- Deep Learning
- Object Detection
- Jupyter Notebook

## Project Structure

Welding-defect-object-detection
